# Маршруты знаний (orchestrator)

Этот **файл — ядро** маршрутизации: **его правит только владелец** (в редакторе, Termux, Cursor).
Агент **не может** писать в `projects/_index.md` — в другие заметки пишет по правилам ниже.
В **Memory Atlas** (веб, ссылка из `/web` с токеном) этот файл можно **редактировать вручную** в панели справа.

Ниже: один путь `memory/notes/…` в строке, затем `|`, затем ключевые слова через запятую.
Пустые строки и строки, начинающиеся с `#`, игнорируются.
Для **других** путей (не `_index.md`) несуществующий файл агент может создать через `write_note`.

# Примеры (замените на свои и домены):

# projects/dinodark.md | dinodark.ru, динодарк, переговоры, аудио
# projects/pupumpa.md | pupumpa.ru, пупумпа, курсы
# marketing/strategy.md | маркетинг, воронка, ценностное
